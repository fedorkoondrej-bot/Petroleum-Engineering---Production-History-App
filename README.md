# Petroleum Engineering – Production History App

A Python-based desktop application for visualizing and analyzing oil well production history data.

Built as a portfolio project to demonstrate data engineering and analytics skills in the oil & gas sector.

> **Note:** This app uses dummy/synthetic data only. No real production data is included.

---

## What it does

- Loads well production data from Excel (`.xlsx`)
- Visualizes production history per well (oil, gas, water rates over time)
- Calculates basic well performance metrics
- Provides a simple GUI for filtering and exploring production records

---

## Tech stack

- **Python** – core logic
- **pandas** – data processing
- **matplotlib** – production charts
- **openpyxl** – Excel file handling
- **tkinter** – GUI interface

---

## Files

| File | Description |
|------|-------------|
| `Production_app.py` | Main application script |
| `10_Wells_Production_Data.xlsx` | Sample dataset (10 synthetic wells) |
| `requirements.txt` | Python dependencies |

---

## How to run

```bash
pip install -r requirements.txt
python Production_app.py
```
